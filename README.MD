# 🚢 Titanic Data Analysis | Power BI Project

## 📊 Overview
This project explores the famous Titanic dataset using **Power BI** to understand the survival patterns of passengers. The focus is on data cleaning, transformation, and building an interactive dashboard to extract meaningful insights.

---

## 🎯 Objective
To analyze passenger survival trends based on different factors such as:
- Gender
- Age
- Passenger Class
- Fare
- Embarkation Port

---

## 📁 Dataset Information
The dataset used is the Titanic dataset (commonly available on Kaggle), which includes:
- Passenger details
- Survival status (0 = Did not survive, 1 = Survived)
- Demographic and travel information

---

## 🧹 Data Cleaning (Power Query)
The following preprocessing steps were performed:
- Handled missing values in Age and Embarked columns  
- Removed or replaced null entries  
- Standardized data formats  
- Created structured and analysis-ready dataset  
- Applied transformations using Power Query Editor  

---

## 📊 Dashboard Features
The Power BI dashboard includes:
- 📌 Survival rate overview (KPI cards)
- 👩‍🦰 Gender-wise survival analysis  
- 🎟️ Passenger class (Pclass) survival comparison  
- 📊 Age distribution of survivors  
- 💰 Fare vs Survival insights  
- 🚢 Embarkation port analysis  
- 🔍 Interactive filters and slicers  

---

## 📈 Key Insights
- Females had a significantly higher survival rate than males  
- First-class passengers had better survival chances  
- Younger passengers were more likely to survive  
- Fare and passenger class were strong indicators of survival  
- Embarkation point showed slight variation in survival patterns
